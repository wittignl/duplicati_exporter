# Duplicati Exporter

Prometheus exporter for Duplicati reports.

## Running

    npm install
    node duplicati_exporter
    
## Usage

    Usage: duplicati_exporter [options]
    
    Options:
      -p, --port  Server port                                        [default: 9118]
      -h, --help  Show help                                                [boolean]
    
    Examples:
      duplicati_exporter -p 9010                      Use port 9010
